# Source Boundary

## Included

This package includes the published paper reference and the Figure 2 source document in original, modernized, rendered, and text-extract forms.

## Excluded

Manuscript drafts, review reports, response-to-review documents, duplicate article PDFs, and broader unrelated congestion-pricing folders are excluded. They are not data or code for the reproducibility package.

No runnable simulation code is included because none was present in the inspected source folders. The paper's equations and numerical descriptions remain the primary reproducibility record.
